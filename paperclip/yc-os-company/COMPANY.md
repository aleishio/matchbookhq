# YC OS

YC OS is a public-data-only development company for building apps and tools around public YC/company information.

Rules:

- Treat the repo as public from day one.
- Use public data only.
- Store real deployment secrets only in Vercel or local dev-only secret files.
- Query gbrain before large planning, debugging, refactors, or architecture decisions.
- Deploy apps through Vercel, not directly from the development VPS.
